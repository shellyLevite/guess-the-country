# Claude / GitHub Copilot Configuration

## Project: Guess The Country

### Rules

- Always use **TypeScript** — never generate plain `.js` files for source code
- Backend: **Fastify** only — do not suggest Express or other frameworks
- Frontend: **React + Vite** — functional components with hooks only, no class components
- Use `async/await` throughout — never use raw `.then()` chains or callbacks
- Keep each file **small and focused** — one component or one concern per file
- API routes go in `server/src/routes/` — never inline routes in `index.ts`
- All types go in `server/src/types.ts` or co-located `.types.ts` files
- CSS: minimal, dark theme, no external UI libraries (no Tailwind, no MUI)
- After each logical step, commit with a **meaningful, lowercase commit message**
- Never add unused imports or dead code
